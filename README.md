# Treino Nuvem ☁️💪

Aplicativo progressivo (PWA) para gestão de treinos focado em simplicidade e performance.

## 🚀 Funcionalidades
- **Gestão de Rotinas:** Crie e organize seus treinos (A, B, C, etc).
- **Registro de Sessão:** Log de séries com peso, repetições e RPE.
- **Dashboard de Evolução:** Gráficos de carga e histórico de recordes (PRs).
- **Medidas Corporais:** Acompanhamento de peso e medidas com histórico.
- **Catálogo de Exercícios:** Biblioteca personalizada para padronizar treinos.
- **Exportação/Importação:** Suporte total a JSON e integração com planilhas (CSV/Excel).
- **Analista IA:** Gerador de prompts otimizados para análise no Google Gemini.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Design Responsivo & Variáveis)
- JavaScript Vanilla (Lógica e Persistência Local)
- Chart.js (Visualização de dados)
- PWA (Manifest & Service Worker para uso offline)

## 📱 Como usar no celular
1. Acesse o link do projeto pelo navegador do celular.
2. Clique em "Adicionar à tela de início".
3. Use como um aplicativo nativo, sem precisar de loja.
